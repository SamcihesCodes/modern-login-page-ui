# Modern Giriş Ekranı (Login Page)

Bu proje, **HTML**, **CSS** ve **PHP** kullanılarak tasarlanmış modern bir giriş ekranı örneğidir.

## Özellikler
* **Glassmorphism** tasarım dili ile modern görünüm.
* Tamamen **Responsive** (mobil uyumlu) yapı.
* Saf HTML ve CSS ile hazırlanmış temiz arayüz.
* PHP entegrasyonu ile temel form yapısı.

## Kullanılan Teknolojiler
* HTML5
* CSS3
* PHP
